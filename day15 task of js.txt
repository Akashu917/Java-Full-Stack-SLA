1. What is JavaScript?
Ans:

	JavaScript is a programming language used to make web pages interactive. It runs in the browser and allows you to:

	Handle user actions (clicks, typing)
	Update content dynamically
	Control multimedia, animations, and more

Simple JavaScript Script:

	alert("Hello, this is a simple JavaScript script!");
