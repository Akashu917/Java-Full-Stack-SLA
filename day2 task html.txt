1.Block-level vs Inline Elements
---> Block-level Elements
Ans :
	Take the full width of the page.
	Always start on a new line.
	Used to structure the page layout.

Examples:

<div>, <p>, <h1> – <h6>

Example:
	<div>
	    <p>This is a paragraph</p>
	    <h1> this my heading </h1>
	</div>
	

---> Inline Elements
Ans :
	Take only the width needed.
	Do not start on a new line.
	Used inside block elements to style or link text.

Examples:
	<span>, <a>

Example:
	<h1>Welcome to the <span style="color:blue;">SLA</span> institute</h1>


2).Basic HTML Tags

1.Headings:

	<h1>Main Heading</h1>
	<h2>Sub Heading</h2>
	<h3>Section Heading</h3>
	<h4>Smaller Heading</h4>
	<h5>Very Small Heading</h5>
	<h6>Smallest Heading</h6>


2.Paragraph:

	<p>This is a paragraph of text</p>


3.Division:

Ans: Used to group content.

	<div>
  	   <p>This paragraph is inside a div</p>
	</div>

4.Span:

Ans: Used for styling part of text.

	<p>This is a <span style="color:blue;">SLA</span>institute</p>
