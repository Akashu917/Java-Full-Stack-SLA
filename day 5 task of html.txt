1. HTML Attributes?

Ans:  Attributes provide extra information about an HTML element.
      They are placed inside the opening tag.

Example:

	<a href="https://www.google.com">Visit Google</a>

	href → attribute

"https://www.google.com" → value

Common Attributes:
	Attribute:	Purpose:
	href	 	link destination
	src		image/file source
	alt		alternative text for images
	title		tooltip text
	class		group elements for styling
	id		unique identifier

2. Using class for Multiple Elements?

Ans: A class can be used on many elements.

HTML code:
<style>
  .highlight{
  color: red;
  font-weight: bold;
}
</style>

<p class="">First class stand</p>
<p class="">Second class stand</p>
<p class="">Third class stand</p>

3. Using id (Unique)

Ans: id must be unique on a page.

HTML code:
<style>
	#main{
  	color: blue;
	}
</style>
<h1 id="main">My Website</h1>

Used often for:

	JavaScript targeting
	page navigation

4. Special Attributes?

Ans: Title (Tooltip)
	<p title="This appears when you hover">Hover over me</p>

Ans: Lang
	<html lang="en">

Ans: Style
	<p style="color:green;">Green text</p>

Ans: Hidden
	<p hidden>This is hidden</p>


5. Creating Hyperlinks?

Ans: Absolute Link (External Website)
	<a href="https://www.wikipedia.org">Open Wikipedia</a>

Ans: Relative Link (Local File)
	<a href="about.html">About Page</a>

6. Extra Link Attributes?

Ans: Open in New Tab
	<a href="https://www.google.com" target="_blank">Google</a>

Ans: Download File
	<a href="file.pdf" download>Download PDF</a>

Ans: Email Link
	<a href="mailto:someone@example.com">Send Email</a>

7. Image Attributes?

Ans: Basic Image
	<img src="cat.jpg" alt="A cute cat">

	src → image location

	alt → text if image fails or for accessibility

8. Image Dimensions?

Ans: <img src="cat.jpg" alt="A cute cat" width="300px" height="200px">

Clickable Image?

Ans: Image inside a link:

	<a href="https://www.google.com">
	<img src="logo.png" alt="Google logo"></a>
